# Joanna Bassey - Portfolio Website

A modern, responsive portfolio website built with Next.js, featuring a sleek black and gold design with smooth animations.

## Features

- **Modern Design**: Clean black background with elegant gold accents
- **Smooth Animations**: AOS (Animate On Scroll) library for engaging user experience
- **Responsive Layout**: Mobile-first design that works on all devices
- **Interactive Components**: Animated counters, project carousel, and contact form
- **Real Projects**: Showcases actual deployed projects with live links

## Tech Stack

- **Framework**: Next.js 16.0.10 with TypeScript
- **Styling**: Tailwind CSS v4
- **Icons**: React Icons
- **Animations**: AOS (Animate On Scroll)
- **Form Handling**: Formspree integration
- **Deployment**: Vercel

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the portfolio.

## Project Structure

```
app/
├── components/
│   ├── Hero.tsx          # Landing section with introduction
│   ├── Experience.tsx    # Animated experience counters
│   ├── About.tsx         # About section with profile
│   ├── Skills.tsx        # Technical skills with icons
│   ├── Projects.tsx      # Project carousel showcase
│   ├── Contact.tsx       # Contact form with Formspree
│   └── Footer.tsx        # Social links and copyright
├── globals.css           # Global styles and animations
├── layout.tsx           # Root layout component
└── page.tsx             # Main portfolio page
```
